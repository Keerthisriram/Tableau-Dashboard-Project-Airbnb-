# Tableau-Dashboard-Project-Airbnb

## Project Objectives
This project aims to analyze Airbnb data using Tableau to uncover market trends, pricing patterns, and geographic demand hotspots. By visualizing key metrics such as occupancy rates, it provides actionable insights to help stakeholders make informed, data-driven decisions.

## Dataset Used
- <a href="https://github.com/Keerthisriram/Tableau-Dashboard-Project-Airbnb-/blob/main/Tableau%20Project%20file.xlsx">Dataset</a>
- <a href ="https://github.com/Keerthisriram/Tableau-Dashboard-Project-Airbnb-/blob/main/Tableau%20Dashboard.png"> Dashboard Interaction</a>

## Key Performance Indicators(KPI)
1.Which zip codes are charges the most in Airbnb.
2.Which year is best for renting out Airbnb( Highest revenue per year in Airbnb)
3.Which bedrooms brings more money?
4.Distinct Count of available bedrooms?
5.Show the Price differences for each locations?

## Process:

- Upload excel sheet into tableau and combined all together using Union.
- Analyse the data and create the measures and dimensions for created dataset
- Create the visuals using dataset in tableau.
- Create the dashboard sheet and Combined all visuals into one sheet.

## Conclusion:

The Airbnb Tableau project successfully transformed complex data into clear, interactive visual insights. It revealed key trends in pricing, demand, and location performance, enabling smarter decision-making. Overall, the analysis empowers stakeholders to optimize strategies, improve guest satisfaction, and enhance profitability.
